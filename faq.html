Frequently Asked Questions

What is vetements?

vetements is an advanced all-in-one Discord bot designed to provide powerful utilities directly in your DMs and servers. No need to download external tools or visit multiple websites - everything is accessible through Discord commands.

How do I use the bot?

Simply add vetements to your server or add it to your apps, then use slash commands like /qrgen, /translate, or /cryptoprice anywhere to access various features. The bot provides interactive menus and easy-to-follow instructions for each command.

Does vetements have moderation commands?

Yes! vetements includes a comprehensive set of moderation commands to help manage your Discord server effectively. These commands are designed for server administrators and moderators:

/kick Kick a user from the server
/ban Ban a user from the server
/unban Unban a user from the server
/lock Lock the current channel for everyone
/unlock Unlock the current channel for everyone
/role Add a role to a user
/unrole Remove a role from a user
/timeout Timeout a user
/untimeout Remove timeout from a user
/softban Softban a user to delete their messages (ban and unban)
/giveaway Create a giveaway
/massrole Add a role to all users with a specific role
/massunrole Remove a role from all users who have it
/slowmode Set slowmode for the current channel
/modlogs View moderation logs for a user or channel
/vcmute Mute a user in voice channels
/vcunmute Unmute a user in voice channels
/nick Change a user's nickname
/invites Check how many invites a user has

All moderation commands require appropriate permissions and include confirmation steps to prevent accidental actions. You can find these commands under the "Moderation" category in the commands page.

Is vetements free?

Yes! Most features are completely free to use. We offer a premium tier with enhanced capabilities and limits for users who want to support development and get additional features.

Where can I get support?

Join our Discord server for support, updates, bug reports, and to connect with the community. You can also use the /support command to get a direct link to our support server.

How do I add vetements to my server?

Click the "Add to Discord" button on the homepage or use the /invite command. You'll need "Manage Server" permissions to add the bot.

What permissions does the bot need?

vetements requires basic permissions like Send Messages, Embed Links, and Attach Files. Some commands may require additional permissions which are clearly indicated when you try to use them. For moderation commands, the bot needs appropriate admin/moderator permissions.

Can I use vetements in DMs?

Yes! Most commands work in direct messages. Simply start a DM with vetements and use slash commands as you would in a server. Note that moderation commands only work in servers where you have appropriate permissions.

How often is vetements updated?

We release regular updates with new features, improvements, and bug fixes. Premium users get early access to new features.
